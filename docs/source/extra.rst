How to Cite AFEM
================
The following Bibtex template can be used to cite AFEM in scientific and
engineering publications:

.. code-block:: bash

    @misc{AFEM,
       author = {Trevor Laughlin},
       year = {2020},
       note = {https://github.com/trelau/AFEM},
       title = {AFEM -- Airframe Finite Element Modeler}
    }
